Plane Jumpers

Plane Jumpers now plays like a neon-flight remix of Subway Surfers. Bank a shimmering stunt plane through a pseudo-3D skyline, skim glowing rings, and dodge drones and turbulence pods that streak toward the camera. Procedural lighting, parallax clouds, and speed trails sell the depth without shipping any texture files.

How to launch

  • Install pygame if you do not already have it:
    pip install pygame
  • Run the game from your terminal:
    python plane_jumpers.py

Controls

  • Left / Right (or A / D) — bank across three aerial lanes
  • Space / W / Up Arrow — pop upward to clear low-flying drones
  • Esc / Q — exit instantly
  • Enter — restart after a crash landing
