Plane Jumpers

Strap into your paper aircraft and weave through the clouds in this endless runner inspired by Subway Surfers. The Python edition of Plane Jumpers drops you into a neon jet stream filled with hazards, rings, and lane-switching aerial tricks.

How to launch

  • Install pygame if you do not already have it:
    pip install pygame
  • Run the game from your terminal:
    python plane_jumpers.py

Controls

  • Left / Right (or A / D) — shift lanes
  • Space / W / Up Arrow — burst upward to dodge low clouds
  • Esc — exit the game
Plane Jumpers procedurally renders its sprites, so no additional art assets are needed. Customize the code in plane_jumpers.py to tweak speeds, spawn rates, and visuals.
